¿Qué es Cloud Computing?

El Cloud Computing es la entrega de servicios informáticos a través de Internet, incluyendo almacenamiento, bases de datos, servidores, redes y software.

SaaS, PaaS e IaaS

  • SaaS: Aplicaciones listas para usar, como Gmail o Dropbox.
  • PaaS: Plataformas para desarrollar aplicaciones sin gestionar la infraestructura, como Heroku.
  • IaaS: Infraestructura virtualizada, como servidores y almacenamiento, por ejemplo Amazon EC2.

¿Qué es Virtualización?

Creación de versiones virtuales de recursos físicos para aprovecharlos mejor.

¿Qué es Docker?

Docker permite crear, desplegar y ejecutar aplicaciones en contenedores, facilitando la portabilidad del software.

Anydesk

Herramienta para acceder y controlar remotamente otro ordenador de manera segura y rápida.

TeamViewer

Software de control remoto que facilita soporte técnico y colaboración entre dispositivos.
