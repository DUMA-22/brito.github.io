Conceptos de Cloud Computing y Acceso Remoto

Cloud

Cloud Computing

Cloud Computing: Es la entrega de servicios informáticos (almacenamiento, servidores, bases de datos, redes y software) a través de Internet, permitiendo acceder a recursos sin necesidad de tenerlos físicamente.

SaaS, PaaS e IaaS:
SaaS (Software as a Service): Software disponible por Internet, sin instalar, ejemplo: Google Docs, Zoom.
PaaS (Platform as a Service): Plataforma que permite desarrollar y desplegar aplicaciones, ejemplo: Heroku, Google App Engine.
IaaS (Infrastructure as a Service): Infraestructura como servicio: servidores, almacenamiento y redes virtuales, ejemplo: AWS, Microsoft Azure.

Virtualización: Creación de una versión virtual de un recurso físico, como un servidor o sistema operativo, permitiendo ejecutar varios entornos independientes en un mismo hardware.

Docker: Plataforma que permite crear, desplegar y ejecutar aplicaciones en contenedores, los cuales incluyen todo lo necesario para que el software funcione de manera consistente en cualquier entorno.
